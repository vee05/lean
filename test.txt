把目录变成仓库：	git init
添加到缓冲区： 		git add 文件
提交到仓库：		git commit -m "描述"
查看仓库状态：		git status
查看修改：			git diff 文件
查看版本：			git log --pretty=oneline
版本回退：				git reset --hard 1094a
创建分支:			git branch <name>
切换分支：			git checkout <name>
创建并切换：		git checkout -b <name>
查看分支：			git branch
删除分支：			git branch -d <name>
合并：				git merge <name>


管理远程git:		git remote add origin git@server-name:path/repo-name.git
推送最新修改:		git push origin master	
克隆：				git clone git@github.com:michaelliao/gitskills.git
